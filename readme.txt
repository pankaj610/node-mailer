Application for sending bulk email.

Souce File: Book2.xlsx 

Step 1: Download and install node.js server from the below link
Link: https://nodejs.org/en/

Step 2: Open command prompt in the root folder of applicaiton.

Step 3: Type the below command to run the program.

Command: node app.js startingRow endingRow
EG: node app.js 3 5

Note: If we don't provide second argument i.e endingRow then email will be sent 
to all the from the startingRow provided.
